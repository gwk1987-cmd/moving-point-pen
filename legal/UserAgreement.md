# 《动点神笔》用户服务协议

**版本更新日期：2026年4月2日**

## 1. 服务的性质与范围
1.1 本平台是一个辅助初中数学教师进行动点问题动画制作的工具。
1.2 本平台利用人工智能（AI）技术解析用户输入的题目文本，并生成对应的几何动画演示。

## 2. 知识产权归属
2.1 **平台所有权**：本平台的软件架构、算法模型（除DeepSeek/JSXGraph等开源或授权组件外）、UI设计、品牌标识及预设的几何模型库，其知识产权均归平台开发者（凯哥/一人公司）所有。
2.2 **用户生成内容（UGC）**：
    - 用户输入的原始文本题目，其知识产权归用户所有。
    - **非付费（免费版）用户**生成的动画视频/GIF，其版权归平台所有，平台授权用户仅用于课堂教学等非营利性场景，且必须保留平台水印。
    - **专业版（Premium）用户**，平台授权其对所生成的无水印/透明背景动画享有商用许可。

## 3. 使用规则与成本管理
3.1 您同意不会通过自动化脚本、爬虫等方式恶意刷取AI解析额度。
3.2 平台有权根据API成本波动调整每日免费解析限额。

## 4. 免责声明（核心保护）
4.1 **AI幻觉风险**：本平台生成的动画由AI解析驱动。由于数学逻辑的复杂性，AI可能在解析过程中产生错误（如速度方向、坐标计算、图形识别错误等）。
4.2 **教师核对义务**：**本平台仅作为辅助工具。用户在正式用于教学、发布或商业用途前，必须对动画的数学准确性进行独立核对。** 平台不承担因动画内容错误而导致的任何教学事故或法律责任。
